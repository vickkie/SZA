**Welcome to the SZA portfolio**, an editorial-style showcase celebrating the artistry of Solána Imani Rowe — known to the world as SZA. Built on the same horizontal-scroll engine as the Viola Wumi portfolio, this site reinterprets the layout for a musician's world: albums, visuals, lyrics, and the cosmic-ocean aesthetic that defines her brand.

## About the Project

A minimalist yet dynamic portfolio that highlights SZA's journey across eras — from _Ctrl_ to _SOS_ and beyond. Each section captures a different facet of her artistry: the writing, the voice, the visuals, and the mythology her fans love.

### Key Features:

- **Dynamic Transitions**: Smooth GSAP-powered animations
- **Interactive Design**: Engaging user experience with minimalist aesthetics
- **Responsive Layout**: Seamless viewing across all devices
- **Performance Optimized**: Fast loading and smooth scrolling

## Tech Stack

- React 17 + TypeScript
- Vite 2
- SCSS modules
- GSAP + Locomotive Scroll
- React Fast Marquee

## Getting Started

```bash
npm install --legacy-peer-deps
npm run dev      # http://localhost:3000
npm run build    # tsc + vite build -> dist/
npm run serve    # preview the build
```
